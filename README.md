# StoryEngine

An Object Oriented, Data-driven platform for novel writing inspired by game engines.

![Untitled design (1)](https://user-images.githubusercontent.com/27149047/176504818-10f48e6c-ab4b-4f88-a0e0-a5e6a00d0bf7.png)

## Development

Next.js app with a Convex backend.

```bash
npm install
npm run dev
```

Use `.env.local` for `NEXT_PUBLIC_CONVEX_URL` (and other public env vars your host needs). Deploy the Convex functions with `npx convex deploy` and configure deployment env vars via `npx convex env set`.
